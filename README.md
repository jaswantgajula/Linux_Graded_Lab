# Linux Graded Lab Assignment

## Repository Overview

This repository contains the solutions for the Linux Graded Lab Assignment (Modules 1–4). The assignment demonstrates the use of Linux commands for system administration, file management, permissions, file links, file I/O, and storage analysis.

## Repository Structure

Linux_Graded_Lab/
│
├── Question1/
├── Question2/
├── Question3/
├── Question4/
└── Question5/

## Contents

### Question 1 – Linux Environment Verification
- Verified user account details
- Displayed user groups
- Checked current shell
- Displayed working directory
- Listed files and directories
- Verified network connectivity

### Question 2 – Secure Project Workspace Setup
- Created a secure project workspace
- Configured directory permissions
- Checked ownership details
- Verified umask value
- Explained permission security

### Question 3 – File System and Link Analysis
- Created a file
- Created hard and symbolic links
- Compared inode numbers
- Observed behavior after deleting the original file
- Collected file metadata

### Question 4 – File Access and I/O Investigation
- Investigated open files
- Examined file descriptors
- Demonstrated output and error redirection
- Checked system resource limits
- Explained Linux file I/O management

### Question 5 – Storage Health Assessment
- Listed available storage devices
- Displayed mounted file systems
- Checked disk usage
- Examined inode utilization
- Documented storage health observations
- Created the report using the vi editor

## Files Included in Each Question

- Report (.txt)
- commands.txt
- Screenshots
- Supporting files (where required)

## Platform Used

Operating System: macOS
Terminal: zsh

Note: Since this assignment was completed on macOS, Linux-specific commands such as `lsblk` were replaced with the equivalent macOS command `diskutil list`.
